## About
Brew Haven is a simple web project created to fulfill the requirements of the course **Web Development - 1: Web Design for Beginners** offered by **CODL, University of Moratuwa**.

## Technologies Used
- **HTML** for structuring the web pages  
- **CSS** for styling and responsive design  
- **JavaScript** for interactivity, including smooth scrolling, form validation, and dynamic content  

## Features
- Interactive coffee menu and image gallery  
- Newsletter subscription form with validation and local storage  
- Smooth scrolling and hover animations  
- Fully responsive layout for different screen sizes
